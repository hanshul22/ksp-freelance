# KSP Hydro Engineers Website

A modern, professional website for KSP Hydro Engineers - an industrial water engineering company specializing in water purification, wastewater treatment, and fire-fighting systems.

## Features

### Navbar
- Fixed floating navbar with rounded white container
- Responsive design with mobile menu
- Logo with gradient background
- Navigation links: Home, About, Products, Services, Projects
- Highlighted Contact button with blue background

### Hero Section
- Two-column layout with content and visual
- Large bold headline with gradient text effect (blue-to-green)
- Supporting description text
- Two CTA buttons (primary and secondary)
- 3D glossy water droplet illustration with SVG
- Floating bubbles background animation
- Trust indicators: ISO Certified, 20+ Years Experience, 500+ Projects Delivered
- Fully responsive design

## Tech Stack

- React 18
- React Router DOM
- Tailwind CSS
- Lucide React (icons)
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
